### eslint_airbnb
eslint作为前端js开发的代码检查工具，配置选择的是airbnb的基础配置, 和一些小改动。

### 在心理上接受它
[在多人协作的前端团队中，ESlint是非常有必要。同一项目组，互相代码review，多人协同开发同一需求等等，没有ESlint...](https://blog.csdn.net/gbwine/article/details/79180012)

### 使用
1.  在IDE中打开eslint工具
2.  配置根目录下的.eslintrc
